# MyPriceTickerApp

A lightweight **Zepp OS Mini App** that displays live price ticker information directly on your wearable device.

This application is designed for Zepp OS smartwatches and provides quick access to real-time pricing data in a clean, minimal interface optimized for small screens.

Repository:
